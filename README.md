# Summer Internship 2026 Project

## Project Title

**Time-Series Analysis of Global Temperature Data and K-Means Clustering on MNIST Handwritten Digits**

## Overview

This project was completed as part of the **Summer Internship Program 2026** conducted by **IDEAS – Institute of Data Engineering, Analytics and Science Foundation, ISI Kolkata**.

The project consists of two major components:

1. **Time-Series Analysis of Global Temperature Data**
2. **K-Means Clustering on MNIST Handwritten Digits Dataset**

The objective was to apply data preprocessing, exploratory data analysis, time-series techniques, and machine learning algorithms to real-world datasets.

## Part 1: Time-Series Analysis of Global Temperature Data

### Dataset Information

The dataset contains historical monthly global temperature records from climate monitoring organizations.

### Tasks Performed

* Data loading and inspection
* Datetime conversion and sorting
* Missing value analysis
* Feature engineering (Year and Month extraction)
* 12-Month Moving Average calculation
* Last 20 years data filtering
* Source-wise temperature comparison
* Year-wise temperature aggregation
* Month-wise temperature analysis
* Pivot table generation

### Key Insights

* Historical temperature trends were analyzed over multiple decades.
* Seasonal temperature patterns were identified.
* Temperature anomalies from different sources were compared.
* Moving averages were used to highlight long-term climate trends.

---

## Part 2: K-Means Clustering on MNIST Dataset

### Dataset Information

The MNIST dataset contains handwritten digit images ranging from 0 to 9.

### Tasks Performed

* Dataset loading using Scikit-Learn
* Feature and target separation
* K-Means clustering with 10 clusters
* Cluster-to-label mapping
* Performance evaluation using Macro F1 Score

### Key Insights

* Similar handwritten digits were grouped successfully using clustering.
* Cluster labels were mapped to actual digit labels.
* Clustering performance was evaluated using the F1 Score metric.
